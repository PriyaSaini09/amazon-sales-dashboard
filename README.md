# Amazon Sales Dashboard 📊

## 🔹 Project Overview
This project analyzes Amazon sales data using Power BI and MySQL to generate business insights.

## 🔹 Tools Used
- Power BI
- MySQL
- Excel / CSV

## 🔹 Key Features
- Total Revenue KPI
- Total Orders KPI
- Average Order Value
- Cancellation Rate
- Sales Trend Analysis
- Top Product Categories
